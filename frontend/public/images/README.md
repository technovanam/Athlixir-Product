# Images Directory

Place your images here: 

## Required Images

- `hero-bg.jpg` - Hero section background image (recommended: 1920x1080 or higher, dark sports field/grass image)

## Image Guidelines

- Use optimized images (WebP or compressed JPG/PNG)
- Hero backgrounds should be at least 1920x1080 for retina displays
- Keep file sizes reasonable (< 500KB for backgrounds after compression)
